# 贡献指南
感谢你愿意为 FkRmShell 贡献代码或建议！

## 🤝 贡献方式
- 提交 Bug 反馈：请通过 Issues 提交，并附上设备信息和日志
- 提交功能建议：请先开 Issue 讨论，再动手实现
- 提交代码 PR：请遵循以下规范

## 📐 代码规范
- Shell 脚本请保持和项目现有代码一致的风格
- 提交信息格式：`fix: 修复xxx问题` 或 `feat: 新增xxx功能`
- 大改动请拆分成多个小提交，方便 review

## 🚀 PR 流程
1.  Fork 本仓库，创建你的分支
2.  在你的分支上修改代码
3.  提交 PR 到 `main` 分支
4.  等待维护者 review 并合并

## ❌ 不接受的贡献
- 与项目功能无关的改动
- 未讨论过的大型重构
- 破坏现有功能的改动

再次感谢你的贡献！
